<h2>ผมทำโปรเจค React Demo ครับ โดยทำ responsive ด้วยครับในขนาดหน้าจอเล็ก  <br>

<h2>package ที่ต้อง download ***โหลดใน cmd
<li>npm i</li>
<li>npm create vite@latest เลือก React</li>
<li>npm install tailwindcss @tailwindcss/vite</li>
<li>npm install react-router-dom</li>
<li>npm install axios</li>
<li>npm install react-icons</li>
<h2>ขั้นตอนลง Web
<li>npm run build</li>
